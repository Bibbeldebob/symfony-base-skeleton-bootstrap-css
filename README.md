# symfony-base-skeleton-bootstrap-css

## Nutzung als Starterprojekt

Dieses Projekt kann als Grundlage für neue Symfony-Projekte verwendet werden.
Es enthält einen Devcontainer mit einer DB und grundlegende Tools.

Symfony ist mit einigen grundlegenden Bundles bereits eingerichtet.
Weiterhin ist Bootstrap CSS bereits per AssetMapper installiert und eingebunden.
